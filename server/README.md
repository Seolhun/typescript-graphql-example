## Pre-Reqirement
1. `TypeScript`
2. `NodeJS`, `Express`
3. `Vue-Cli`
4. `GraphQL`
5. `json-server`

## Content List
1. `Grammar`, `TypeScript Method`
2. `Design Pattern`
3. `GraphQL`

## How to run
- `Server`
  - `npm install -g ts-node vue-cli typescript`
  1. `npm run dev`
  2. [http://localhost:4000/graphql](http://localhost:4000/graphql)

- `DB`
  1. `npm run json:server`
  2. [http://localhost:3100/users](http://localhost:3100/users)
